Notes:

This dashboard was built using Angular 1.6.7 with the following modules:

  • angular-route
  • angular-toastr
  • ui-bootstrap

The backend uses express, body-parser, log4js, pg and request.

The data store is postgres, I've dumped the schema which can be found in the database directory.

If you want to run it locally you'll need to provide the database connection parameters in the config.js file. I've kept the project as one big monolith rather than splitting into three to make it easier to run up locally. I'm running Node version 6 here.
